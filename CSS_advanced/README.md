# CSS Advanced

This project is part of the Holberton School Web Front-End curriculum.

## Description

The goal of this project is to apply advanced CSS concepts to build a fully styled and responsive website based on a provided HTML structure.  
The project focuses on layout, positioning, typography, responsiveness, and visual consistency across modern browsers.

All styling is done using **pure CSS**, without any external frameworks.

## Learning Objectives

By completing this project, I am able to:

- Understand and apply advanced CSS selectors
- Use CSS properties and values effectively
- Differentiate between block and inline elements
- Apply CSS resets for browser consistency
- Use CSS variables
- Understand CSS specificity and cascade rules
- Work with positioning (relative, absolute, fixed)
- Use flexbox and grid-like layouts
- Handle fonts, icons, and images
- Apply pseudo-classes and pseudo-elements
- Create animations and transitions
- Use transformations (2D and 3D)
- Understand vendor prefixes
- Organize CSS code in a clean and maintainable way
